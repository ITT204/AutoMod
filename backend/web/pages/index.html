AutoMod

The perfect Discord bot to handle all things moderation and beyond!

  • Add To Server

Features

  • Automoderator

    Robust & highly efficient automoderator with customisable rules, filters & punishments.

  • Case System

    Case system for past moderation actions to hold your moderators accountable.

  • Custom Commands

    Create custom slash commands & text based responders, so you don't have to keep repeating yourself.

  • Level System

    The easy-to-use & balanced level system allows you to reward active users that participate in your server.

  • Auto Roles

    Let the bot automatically give users a certain role when joining your server or even setup reaction roles.
